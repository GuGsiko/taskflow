# TaskFlow Frontend

TaskFlow is a minimal task management web application built with React and Tailwind CSS.

## Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Active / Completed)
- Clean and minimal UI
- Component-based architecture

## Tech Stack
- React
- Tailwind CSS
- Vite
